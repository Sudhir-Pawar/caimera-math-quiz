# Caimera Math Quiz

**Play here:** [https://caimera-math.onrender.com/](https://caimera-math.onrender.com/)

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Socket.io-client
- **Backend:** NodeJS, Express, Socket.io, TypeScript
- **Deployment:** Render.com

## Running Locally

To run this project on your local machine, you will need two terminal windows:

**1. Start the Backend:**

```bash
cd backend
npm install
npm run dev
```

(The server will start on port 3000)

**2. Start the Frontend:**

```bash
cd frontend
npm install
npm run dev
```
